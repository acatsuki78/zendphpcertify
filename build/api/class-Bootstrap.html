Class Bootstrap

Bootstrap principal de l'application.

Bootstrap principal de l'application, lancé automatiquement par Zend_Application en complément des initialisations déclarées dans le application.ini :

  • Initialise le cache d'autoloading
  • Initialise l'adaptateur de traduction par défaut
